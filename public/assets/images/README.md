# Assets Directory

## Images Required

### Hero Background Image
- **Filename:** `hero-globe.webp`
- **Description:** A high-quality image of a digital globe or network visualization with cyber/tech aesthetic
- **Recommended specs:**
  - Format: WebP (for optimal performance)
  - Dimensions: 1920x1080px or larger
  - File size: < 200KB (optimized)
  - Style: Dark, futuristic, with network connections or digital elements
  - Color scheme: Should work well with dark backgrounds and purple accents

### How to Add the Hero Image

1. Find or create a suitable cyber globe/network image
2. Convert to WebP format using tools like:
   - Online: https://squoosh.app/
   - CLI: `cwebp input.png -q 80 -o hero-globe.webp`
3. Place the file in this directory (`assets/images/hero-globe.webp`)

### Alternative Image Sources

- **Unsplash:** Search for "cyber network" or "digital globe"
- **Pexels:** Search for "technology network" or "digital world"
- **Generate with AI:** Use MidJourney, DALL-E, or Stable Diffusion with prompts like:
  - "futuristic digital globe with network connections, cybersecurity theme, dark background, purple accents, 4k"
  - "abstract cyber network visualization, dark blue and purple, global connectivity, technology"

### Fallback
If you don't have the image yet, the page will still work. The background will use the gradient and particle effects. The image adds visual depth but is not required for functionality.
